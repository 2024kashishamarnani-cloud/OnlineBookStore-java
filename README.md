# Online Bookstore Management System

A Java Swing + MySQL application for managing books, users, and orders.

## Features
- Customer login and book browsing
- Add to cart and checkout system
- Review and payment system
- Real-time database sync with MySQL
- Admin management

## Tech Stack
- Java (Swing, JDBC)
- MySQL (via XAMPP)
- NetBeans IDE

## Database Setup
1. Open phpMyAdmin
2. Create database: `bookstore_db`
3. Import `database.sql`

## Run
- Open project in NetBeans → Run `LoginFrame.java`
- OR run the standalone `.jar`
